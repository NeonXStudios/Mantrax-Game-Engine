Getting Started

Welcome to the Getting Started guide! This section will help you set up and get started with the platform.

Step 1: Installation

To begin, you must have the engine installed. First, close this documentation page and use the options on the right side to click on Download Engine. The hub will automatically install the engine for you.

  • Visual Studio 2022 with C++ and MSBuild components
  • Visual Studio Code (VSCode)

Step 2: Setting Up

After downloading, extract the files into your desired directory. Make sure you have Visual Studio 2022 and VSCode installed as prerequisites to compile and edit your projects.

Need Help?

If you encounter any issues, feel free to check the documentation or contact support.
