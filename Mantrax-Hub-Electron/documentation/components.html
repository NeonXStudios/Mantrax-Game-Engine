Components Table

This table provides an overview of the available components and their functionalities.

Component Name Description
ModelComponent Handles the 3D model representation of the entity.
GMaterial Manages the material properties like textures and shaders.
GBody Defines the physical body attributes for physics simulations.
GCollision Handles collision detection and response.
GScript Integrates scripting capabilities for entity behavior.
GAudio Manages audio components for sound effects and music.
GCharacter Defines attributes and behavior for character entities.
GCamera Handles the camera view and perspective settings.
GAnimator Controls animations for models and characters.
