Introducción a MantraxEngine 2D

Bienvenido a la documentación de MantraxEngine 2D, un motor de juegos 2D construido en C++.

Instalación

Para comenzar a usar MantraxEngine 2D, sigue los siguientes pasos:

  • Clona el repositorio desde GitHub.
  • Abre el proyecto en tu IDE favorito (recomendamos Visual Studio).
  • Compila el proyecto y empieza a programar tus juegos.

Ejemplo de Código

Este es un ejemplo de cómo crear un objeto en el motor:


                        #include <iostream>
                        #include "MantraxEngine.h"
                        
                        int main() {
                            MantraxEngine::init();
                            MantraxEngine::createObject("Player", 50, 50);
                            std::cout << "Juego inicializado" << std::endl;
                            return 0;
                        }
